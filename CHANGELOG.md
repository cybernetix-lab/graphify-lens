# Changelog

All notable changes to this project will be documented in this file.

## [0.1.0] — 2026-06-01

### Added
- Knowledge graph parser with support for 8 node types (concept, topic, runbook, decision, case, faq, article, source)
- Interactive D3.js force-directed graph visualization with zoom, drag, and tooltip
- Five-dimension quality assessment engine (Coverage, Accuracy, Freshness, Governance, Reuse & Growth)
- Quality dashboard with real-time metrics and historical trend charts (Chart.js)
- Background scheduler with configurable Git auto-commit
- Cross-platform build support (macOS, Linux, Windows; amd64, arm64)
- Single-binary distribution with embedded frontend via Go embed
- REST API with 6 endpoints
- Sample knowledge base data
